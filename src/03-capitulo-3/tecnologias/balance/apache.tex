\subsubsection{Apache}
\label{soa:tecnologias:apache}

Apache es un servidor web que surgió en 1995 y desde 1999 es desarrollado por la \eng{Apache Foundation}, fundación a la dio origen. Desde su creación y durante muchos años, este producto fue el estándar \textit{de facto} a la hora de instalar servidores web.


Ver \url{http://httpd.apache.org/docs/2.2/mod/mod_proxy_balancer.html}
